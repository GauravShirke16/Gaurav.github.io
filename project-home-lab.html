Home Security Lab Setup

Status: Planned Network Security Virtualisation Ethical Hacking Tools: VirtualBox · Kali Linux · Wireshark
This project is planned and will be documented here once completed. The structure below outlines the full methodology I will follow.
Overview

A home security lab gives you a safe, isolated environment to practice real-world attack and defence techniques without risking any live systems. Using free virtualisation software, I will build a mini network of virtual machines — one acting as the attacker, others as vulnerable targets.

This lab directly mirrors the kind of environments I work in professionally — hybrid infrastructure with multiple network zones — and will help me practise the same techniques used in my day job at Essen Vision.

Step-by-step methodology
01
Install VirtualBox
Download and install Oracle VirtualBox (free). This creates a "hypervisor" — a program that runs virtual computers inside your real computer.
02
Set up Kali Linux VM (attacker)
Download the Kali Linux ISO and create a VM with 4GB RAM. Kali is the industry-standard ethical hacking OS with 600+ pre-installed security tools.
03
Set up vulnerable target VMs
Install Metasploitable 2 (intentionally vulnerable Linux) and DVWA (Damn Vulnerable Web App) as practice targets. Both are free and legal to attack.
04
Configure an isolated internal network
Create a VirtualBox "Host-Only" network so the VMs can communicate with each other but are completely isolated from the real internet.
05
Run attacks and document findings
Perform network scans (nmap), exploit known vulnerabilities (Metasploit), capture traffic (Wireshark), and document every step as a formal security report.
06
Apply defensive hardening
After each attack, apply the appropriate fix — firewall rules, patch the vulnerability, restrict access — then re-test to confirm the control works.
